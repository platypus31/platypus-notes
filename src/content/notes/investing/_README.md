## 🚀 新增一篇文章

1. **複製模板**：在 Obsidian 打開 `publishing/_TEMPLATE.md` → 右鍵 `Make a copy` → 移到 `investing/` 資料夾
2. **改檔名**：用英文 slug，例如 `kilimanjaro-gear-list.md`
3. **改 frontmatter**：
   ```yaml
   topic: investing      ← 這個最重要
   title: "..."
   date: 2026-04-22
   ```
4. **寫內容**
5. **存檔 → 告訴 Claude「發布」**
